  1.  trait Reducer[M <: Model, S, R] extends AnyRef

    A pipeline component that takes a Model.Reaction (wrapped into an InterpreterResult ) and returns a Model.State wrapped into an ReducerResult.

    A pipeline component that takes a Model.Reaction (wrapped into an InterpreterResult ) and returns a Model.State wrapped into an ReducerResult. The execution cannot fail.

    M

    The concrete type of the Model in use.

    S

    The concrete type of the Model.State in use. It should be derived from M.

    R

    The concrete type of the Model.Reaction in use. It should be derived from M.

    A wrapper for the output of the Reducer execution. It should contain a Model.State instance.

    The Model.State concrete type.

Value Members

  1.  object Reducer

    Companion object for the Reducer trait.

    Companion object for the Reducer trait. It exposes some utilities to instantiate the Reducer with the right constraints between types.

    It exposes an Reducer::apply() to build an Reducer with the right types, as some constraints between the used types are defined. In addition, it provides a Reducer::builder() utility used into the pipeline to inject the Model.State in a separate phase, compared to passing the state directly to Reducer::reduce().
