SimpleOpenable

case class SimpleOpenable(requiredKey: Option[OpenableExt.Key] = None, onOpenExtra: OpenableExt.Reaction = Reaction.empty)(implicit subject: OpenableExt.I) extends Openable with Product with Serializable

Standard implementation of the Openable behavior.

This is a behavior associated to an Item that can be opened a single time. The standard Reaction consists into changing an internal variable that indicates the "openness" of the subject, when Statement "open subject (with something)" is called.

requiredKey

The Key required to open the Item. If not passed, the Item does not require a Key to be opened.

onOpenExtra

Reaction to be executed into the State when opened, after the standard Reaction. It can be omitted.
