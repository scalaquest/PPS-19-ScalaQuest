SimpleRoomLink

case class SimpleRoomLink(endRoomRef: RoomRef, endRoomDirection: Direction, openable: Option[RoomLinkExt.Openable] = None, onEnterExtra: RoomLinkExt.Reaction = Reaction.empty)(implicit subject: RoomLinkExt.I) extends RoomLink with Delegate with Product with Serializable

Standard implementation of RoomLink.

endRoomRef

The RoomRef of the Room that will be set as the new location of the player, after entering the subject.

endRoomDirection

The Direction that will be associated to the end room, when available to enter.

openable

If the subject have to be opened before entering, the RoomLink should include Some Openable behavior here. If the subject is set as always open, it can be set to None.

onEnterExtra

Reaction to be executed into the State when entered, after the standard Reaction. It can be omitted.
