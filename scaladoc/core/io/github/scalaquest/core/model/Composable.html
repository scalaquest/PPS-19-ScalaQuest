Composable

trait Composable[A] extends TriggerPusher[A]

A mixin for TriggerPusher s, that enables the possibility to define two levels of MessageTriggers.

This is commonly used when it is needed to define some base extensible triggers, that should be easily extended letting the user defining the extras. When the triggers are effectively used, extras will be privileged, as the extra should always overwrite the base.
