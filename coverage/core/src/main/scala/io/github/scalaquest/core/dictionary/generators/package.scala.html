6 /**
7  * This package provides two type classes `Generator` and `GeneratorK`. They are composable wrappers
8  * around functions. In addition, there are facility methods that allow the creation of these type
9  * classes.
10  *
11  * For example:
12  * {{{
13  *   implicit val generator = Generator.makeEntry((x: Int) => x -> Math.pow(x, 2))
14  *   val generatorK = new GeneratorK[List, Int, Map[Int, Double]]
15  * }}}
16  *
17  * A [[GeneratorK]] can create a Map from a List.
18  * {{{
19  *   generatorK.generate(List(1, 2, 3))

23  * {{{
24  *   Map(
